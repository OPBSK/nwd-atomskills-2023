Поставка состоит из mail_service.exe, Connection.udl, libeay32.dll, ssleay32.dll
1. Откройте файл конфигурации Connection.udl двойным кликом или с помощью текстового редактора
2. Если открыли файл с помощью двойного клика, то во вкладке Соединение в поле "Выберите или введите имя сервера", укажите имя(адрес) сервера базы данных
	Установите флажок в "следующие имя и пароль пользователя" в пункте "Для входа в сервер использовать"
3. В пункте "Выберите базу данных на сервере" выберите вашу базу данных (напр. AS2022)
4. Нажмите ОК

5. Если вы открыли файл с помощью текстового редактора, то вам наобходимо отредактировать поля "Initial Catalog=db_name", вместо db_name впишите имя вашей базы данных
В поле "Data Source=server_name", укажите вместо server_name IP-адрес вашего сервера базы данных или его имя

6. Проверьте работоспособность файла дабл кликом, если он сломался после редактирование через текстовый редактор, 
то удалить этот файл и создайте его заного с именем Connection.udl и выполняйте пункты 1, 2, 3, 4.

7. После включите приложение с помощью файла mail_service.exe 
8. Авторизуйтесь на сервере БД c помощью данных администратора и в почтовом сервисе Gmail
	Логин почты - techlifeas@gmail.com
	Пароль приложения почты - rhvzrvenmgkccdzb (учтите, что это не пароль от аккаунта почты, а пароль для приложений)
	Пароль от почты - ms12d7fV (Используется для входа в почту, не для авторизации в приложении)
9. После авторизации сервер сообщит о том, что он активен.
10. Сервер выполняет проверку баз и почтового ящика каждые пять минут, 
 либо вы можете форсировать проверку с помощью кнопок в блоке управления (учтите, что письмо на почту приходит не сразу)
