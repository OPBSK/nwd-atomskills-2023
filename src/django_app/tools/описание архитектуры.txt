Основная часть информационной системы AUMANU реализована с использованием современных web-технологий:
Python, фреймворк Django, Bootstrap, мессенджер Telegramm, Embarcadero Delphi.
Реализована трехзвенная инфраструктура, с использованием базы данных.
Для связи с CRM системой используется микросервис rest_sender, кэширующий набор данных в базе MSSQL и 
обеспечивающий двустроннюю связь.
Связь с Telegramm обеспечивается встроенным в Web приложение модулем,
а связь с почтовым сервисом Gmail - микросервисом mail_server.
